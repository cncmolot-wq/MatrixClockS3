# Weather Copy RU

Рабочий словарь и текстовые заготовки для погодного интерфейса `MatrixClock`.

Назначение:
- держать все пользовательские погодные фразы в одном месте;
- править формулировки без копания по коду;
- использовать как источник для `ticker`, карточек, Web UI и future runtime;
- удобно отдавать в ChatGPT для языковой и стилистической шлифовки.

Правило стиля:
- короткие законченные фразы;
- без машинописи;
- без техничных кодов в пользовательском интерфейсе;
- без сокращений там, где лучше читается полное слово;
- дружелюбно и бытово, а не “метеостанция для инженера”.

## Заголовок

- выводить бегущей строкой, без символов миниатюр. 
- `---Имя--- ---Дата--- Погода`
- пример: - `Кривой Рог Днепропетровская область Украина 26.04.26г Погода: Ясно Температура +11° Ветер северный 5 км/ч Давление 758 мм рт.ст.
